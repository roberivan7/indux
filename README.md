# 🏭 Infra Shine Guard

Sistema de **Monitoramento e Controle Industrial** desenvolvido como projeto integrador do **SENAI**, com foco em supervisão industrial, monitoramento de equipamentos e gerenciamento operacional.

---

## 📌 Sobre o Projeto

O **Infra Shine Guard** é uma solução voltada para ambientes industriais, permitindo o monitoramento e controle de equipamentos em tempo real.

O sistema foi desenvolvido com o objetivo de simular uma aplicação próxima da realidade empresarial e industrial, aplicando conceitos de:

- Desenvolvimento Web
- Banco de Dados
- Integração PHP + MySQL
- Modelagem de Sistemas
- Controle de Processos Industriais

![Logo do Projeto](./img/logopn2.png)  

---4

## 🎯 Objetivo

Criar um sistema capaz de realizar:

- Supervisão industrial
- Controle de processos
- Monitoramento de temperatura
- Monitoramento de pressão
- Gerenciamento de alarmes industriais

Além disso, o sistema permite acompanhar o estado operacional dos equipamentos monitorados.

# 🏗️ Estrutura do Negócio

O projeto foi desenvolvido considerando diferentes modelos de operação empresarial.

---

## 📦 Produto Físico

Controle de itens físicos contendo:

- Estoque
- Quantidade
- Entrada e saída

### Exemplos

- Sensores
- Motores
- Fios
- Componentes industriais

---

## 🛠️ Serviço

Controle de prestação de serviços técnicos contendo:

- Profissionais
- Agenda
- Disponibilidade

### Regra de Negócio

O sistema impede que o mesmo profissional seja alocado em dois serviços simultaneamente.

### Exemplos

- Técnicos industriais
- Manutenção
- Mão de obra especializada

---

## 🚜 Locação

Controle de equipamentos alugados contendo:

- Data de retirada
- Data de devolução
- Disponibilidade

### Regra de Negócio

O sistema impede locação duplicada do mesmo equipamento no mesmo período.

### Exemplos

- Máquinas industriais
- Equipamentos de manutenção
- Ferramentas especializadas
